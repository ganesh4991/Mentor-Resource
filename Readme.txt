Steps to Run the program:
1. Clone the project from here: https://github.com/ganesh4991/sample-web-service
2. Install mvn:
	- For Mac and Linux users : sudo apt-get install maven
3. Run the following command:
	- mvn spring-boot:run

This enables the web application to run on localhost:8080

Database:
The repository contains a sample database dump which you can load into your database


Feature implemented:
1. Register - registers a new user into the database with encryption
2. Graphs - Graphs display static data
3. Ajax Queries to backend

Future Work:
